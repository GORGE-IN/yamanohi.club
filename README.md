# yamanohi.club 2018
===

このプロジェクトは、**2018年8月11日**に[新宿ドゥースラー](http://duusraa.com/)で行われるyamanohi.clubというイベントを紹介するものです。

## Description

このイベントには、以下の出演者がDJもしくはライブを行います。
各出演者の時間は30分もしくは45分です。

- MOODMAN
- 俚謡山脈
- 石井タカアキラ with 渡邊洋輔（石井タカアキラグループ）
- クラーク内藤
- Drastik Adhesive Force
- hanali
- kampingcar
- Indus Bonze
- nomuson
